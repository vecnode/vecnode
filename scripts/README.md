### Windows 11

Backup launcher:
```bat
scripts\win11\main.bat
```

Personal repositories:
```bat
scripts\win11\download_all_repos.bat [username]
```

Organizations:
```bat
scripts\win11\download_all_orgs.bat [org1 org2 org3 ...]
```

Run SilverBullet (Docker):
```bat
scripts\win11\run_silverbullet.bat
```

### Ubuntu 22.04+

Backup launcher:
```bash
./scripts/ubuntu22/main.sh
```

Personal repositories:
```bash
./scripts/ubuntu22/download_all_repos.sh [username]
```

Organizations:
```bash
./scripts/ubuntu22/download_all_orgs.sh [org1 org2 org3 ...]
```

Run SilverBullet (Docker):
```bash
./scripts/ubuntu22/run_silverbullet.sh
```

Stop container:
```bash
docker stop silverbullet-local
```
